еҸҲеҸ«иҷҡжӢҹжһ„йҖ еҮҪж•°(VIrtual Constructor)жЁЎејҸжҲ–иҖ…еӨҡжҖҒжҖ§е·ҘеҺӮжЁЎејҸгҖӮе·ҘеҺӮж–№жі•жЁЎејҸе®ҡд№үдёҖдёӘеҲӣе»әдә§е“ҒеҜ№иұЎзҡ„е·ҘеҺӮжҺҘеҸЈпјҢе°Ҷе®һйҷ…еҲӣе»әжҖ§е·ҘдҪңжҺЁеҮәеҲ°еӯҗзұ»дёӯгҖӮ

иӢұж–ҮеҺҹиҜқпјҡ

Define an interface for creating an object,but let subclasses decide which class to instantiate.Factory Method lets a class defer instantiation to sub classes.

е®ҡд№үдёҖдёӘз”ЁдәҺеҲӣе»әеҜ№иұЎзҡ„жҺҘеҸЈпјҢи®©еӯҗзұ»еҶіе®ҡе®һдҫӢеҢ–е“ӘдёӘзұ»гҖӮе·ҘеҺӮж–№жі•дҪҝдёҖдёӘзұ»зҡ„е®һдҫӢеҢ–е»¶иҝҹеҲ°е…¶еӯҗзұ»гҖӮ

  • з®ҖеҚ•е·ҘеҺӮжЁЎејҸ:дёҖдёӘе·ҘеҺӮзұ»еңЁдә§е“Ғзұ»е®һдҫӢеҢ–зҡ„дёӯеҝғдҪҚзҪ®пјҢе®ғзҹҘйҒ“жҜҸдёҖдёӘдә§е“ҒпјҢе®ғеҶіе®ҡе“ӘдёҖдёӘдә§е“Ғзұ»еә”еҪ“иў«е®һдҫӢеҢ–гҖӮжңүзӮ№пјҡе…Ғи®ёе®ўжҲ·з«ҜзӣёеҜ№зӢ¬з«ӢдёҺдә§е“Ғзҡ„еҲӣе»әиҝҮзЁӢпјҢе№¶дё”еңЁзі»з»ҹеј•е…Ҙж–°дә§е“Ғзҡ„ж—¶еҖҷж— йңҖдҝ®еӨҚе®ўжҲ·з«ҜпјҢеҚіеңЁжҹҗз§ҚзЁӢеәҰдёҠж”ҜжҢҒвҖңејҖ-й—ӯвҖқеҺҹеҲҷгҖӮзјәзӮ№пјҡж”ҜжҢҒвҖңејҖ-й—ӯвҖқеҺҹеҲҷзҡ„еҠӣеәҰдёҚеӨҹпјҢдёҖж—Ұжңүж–°зҡ„дә§е“ҒеҠ е…ҘеҲ°зі»з»ҹдёӯпјҢе°ұйңҖиҰҒдҝ®ж”№е·ҘеҺӮзұ»пјҢе№¶е°ҶйҖ»иҫ‘еҠ е…ҘеҲ°е·ҘеҺӮзұ»дёӯгҖӮ
  • е·ҘеҺӮж–№жі•жЁЎејҸ:жҳҜеҜ№з®ҖеҚ•е·ҘеҺӮжЁЎејҸзҡ„иҝӣдёҖжӯҘжҠҪиұЎгҖӮйҒҝе…ҚдәҶе…¶зјәзӮ№пјҢдҝқз•ҷдәҶдјҳзӮ№гҖӮж ёеҝғзҡ„е·ҘеҺӮзұ»дёҚеҶҚиҙҹиҙЈжүҖжңүдә§е“Ғзҡ„еҲӣе»әпјҢиҖҢжҳҜе°Ҷе…·дҪ“зҡ„еҲӣе»әе·ҘдҪңдәӨз»ҷе…¶еӯҗзұ»гҖӮиҝҷдёӘж ёеҝғзұ»жҲҗдёәдёҖдёӘжҠҪиұЎе·ҘеҺӮзҡ„и§’иүІпјҢд»…иҙҹиҙЈз»ҷеҮәе…·дҪ“е·ҘеҺӮеӯҗзұ»еҝ…йЎ»е®һзҺ°зҡ„жҺҘеҸЈпјҢдёҚжҺҘи§Ұз»ҶиҠӮгҖӮ
  • жҠҪиұЎе·ҘеҺӮжЁЎејҸ:е·ҘеҺӮж–№жі•жЁЎејҸжҳҜй’ҲеҜ№дёҖдёӘдә§е“Ғзӯүзә§з»“жһ„пјҢиҖҢжҠҪиұЎе·ҘеҺӮжЁЎејҸеҲҷйңҖиҰҒйқўеҜ№еӨҡдёӘдә§е“Ғзӯүзә§з»“жһ„гҖӮ

е·ҘеҺӮж–№жі•жЁЎејҸзҡ„еӣӣз§Қи§’иүІпјҡ

  • жҠҪиұЎе·ҘеҺӮи§’иүІпјҲCreatorпјүпјҡиҜҘи§’иүІжҳҜе·ҘеҺӮж–№жі•жЁЎејҸзҡ„ж ёеҝғпјҢдёҺеә”з”Ёзі»з»ҹж— е…іпјҢд»»дҪ•еҲӣе»әеҜ№иұЎзҡ„е·ҘеҺӮзұ»йғҪеҝ…йЎ»е®һзҺ°е®ғ.
  • е…·дҪ“е·ҘеҺӮи§’иүІпјҲConcrete CreatorпјүпјҡиҜҘи§’иүІе®һзҺ°дәҶжҠҪиұЎе·ҘеҺӮжҺҘеҸЈпјҢеҗ«жңүдёҺеә”з”ЁеҜҶеҲҮзӣёе…ізҡ„йҖ»иҫ‘пјҢе№¶дё”йҖҡиҝҮи°ғз”Ёеә”з”ЁзЁӢеәҸжқҘеҲӣе»әдә§е“ҒеҜ№иұЎгҖӮ
  • жҠҪиұЎдә§е“Ғи§’иүІпјҲProductпјүпјҡиҜҘи§’иүІиҙҹиҙЈе®ҡд№үдә§е“Ғзҡ„е…ұжҖ§пјҢе®һзҺ°еҜ№дә§е“ҒжңҖжҠҪиұЎзҡ„е®ҡд№үгҖӮ
  • е…·дҪ“дә§е“Ғи§’иүІпјҲConcrete Productпјүпјҡж”№и§’иүІе®һзҺ°жҠҪиұЎдә§е“Ғи§’иүІжүҖеЈ°жҳҺзҡ„жҺҘеҸЈпјҢе·ҘеҺӮж–№жі•жЁЎејҸжүҖеҲӣе»әзҡ„жҜҸдёҖдёӘеҜ№иұЎйғҪжҳҜжҹҗдёӘе…·дҪ“дә§е“Ғи§’иүІзҡ„е®һдҫӢгҖӮ

е·ҘеҺӮж–№жі•жЁЎејҸзҡ„еә”з”Ёпјҡ

дјҳзӮ№пјҡ

  • е°ҒиЈ…иүҜеҘҪпјҢд»Јз Ғз»“жһ„жё…жҷ°гҖӮеҲӣе»әеҜ№иұЎе®ӨеҸӢзәҰжқҹжқЎд»¶зҡ„пјҢеҰӮжһңдёҖдёӘи°ғз”ЁиҖ…йңҖиҰҒдҪҝз”ЁдёҖдёӘе…·дҪ“зҡ„дә§е“ҒеҜ№иұЎпјҢеҸӘйңҖиҰҒзҹҘйҒ“иҝҷдёӘдә§е“ҒеҜ№иұЎзҡ„зұ»еҗҚжҲ–зәҰжқҹеӯ—з¬ҰдёІеҚіеҸҜпјҢдёҚз”ЁзҹҘйҒ“еҲӣе»әеҜ№иұЎзҡ„иҝҮзЁӢеҰӮдҪ•гҖӮйҷҚдҪҺдәҶжЁЎеқ—й—ҙзҡ„иҖҰеҗҲеәҰгҖӮ
  • еҸҜжү©еұ•пјҡеңЁиҰҒеўһеҠ дә§е“Ғзұ»ж—¶пјҢеҸӘиҰҒдҝ®ж”№е…·дҪ“зҡ„е·ҘеҺӮзұ»жҲ–жү©еұ•дёҖдёӘе·ҘеҺӮзұ»е°ұеҸҜд»ҘйҖӮеә”еҸҳеҢ–гҖӮ
  • еұҸи”Ҫдә§е“Ғзұ»пјҡдә§е“Ғзұ»зҡ„е®һзҺ°еҰӮдҪ•еҸҳеҢ–пјҢи°ғз”ЁиҖ…дёҚйңҖиҰҒе…ізі»пјҢеҸӘйңҖиҰҒе…ізі»дә§е“Ғзҡ„жҺҘеҸЈпјҢеҸӘиҰҒжҺҘеҸЈдёҚеҸҳпјҢдёҠеұӮжЁЎеқ—е°ұдёҚйңҖиҰҒеҸҳеҢ–гҖӮеӣ дёәдә§е“Ғзҡ„е®һдҫӢеҢ–жҳҜз”ұе·ҘеҺӮиҙҹиҙЈзҡ„пјҢе…·дҪ“з”ҹдә§дҪ•з§Қдә§е“ҒеҜ№иұЎж—¶з”ұдёҚеҗҢзҡ„е·ҘеҺӮзұ»еҶіе®ҡзҡ„гҖӮ
  • и§ЈиҖҰжЎҶжһ¶пјҡй«ҳеұӮжЁЎеқ—еҸӘйңҖиҰҒзҹҘйҒ“дә§е“Ғзҡ„жҠҪиұЎзұ»пјҢе…¶д»–е®һзҺ°зұ»йғҪдёҚйңҖиҰҒе…іеҝғгҖӮз¬ҰеҗҲиҝӘзұізү№жі•еҲҷгҖҒдҫқиө–еҖ’зҪ®еҺҹеҲҷпјҲеҸӘдҫқиө–дә§е“Ғзұ»зҡ„жҠҪиұЎпјүгҖҒйҮҢж°ҸжӣҝжҚўеҺҹеҲҷпјҲеҸҜд»ҘдҪҝз”Ёеӯҗзұ»жӣҝжҚўзҲ¶зұ»дә§е“ҒпјүгҖӮ

дҪҝз”ЁеңәжҷҜпјҡ

  • е·ҘеҺӮж–№жі•жЁЎејҸжҳҜеҲӣе»әдёҖдёӘж–№жі•зҡ„жӣҝд»Је“ҒпјҢеӣ жӯӨеңЁжүҖжңүйңҖиҰҒз”ҹжҲҗеҜ№иұЎзҡ„ең°ж–№йғҪеҸҜд»ҘдҪҝз”ЁпјҢдҪҶжҳҜйңҖиҰҒж…ҺйҮҚиҖғиҷ‘жҳҜеҗҰйңҖиҰҒеўһеҠ дёҖдёӘе·ҘеҺӮзұ»иҝӣиЎҢз®ЎзҗҶпјҢеўһеҠ д»Јз Ғзҡ„еӨҚжқӮеәҰгҖӮ
  • зҒөжҙ»гҖҒеҸҜжү©еұ•зҡ„жЎҶжһ¶еҸҜдҪҝз”Ёе·ҘеҺӮжЁЎејҸгҖӮ
  • з”ЁеңЁејӮжһ„йЎ№зӣ®дёӯгҖӮеҰӮпјҡйҖҡиҝҮWebServiceдёҺдёҖдёӘйқһjavaйЎ№зӣ®дәӨдә’гҖӮ
  • жөӢиҜ•й©ұеҠЁејҖеҸ‘зҡ„жЎҶжһ¶дёӢдҪҝз”ЁгҖӮеҰӮпјҡжөӢиҜ•зұ»Aе°ұйңҖиҰҒеҗҢж—¶дә§з”ҹзұ»Aе…іиҒ”зҡ„зұ»BпјҢдҪҝз”Ёе·ҘеҺӮж–№жі•жЁЎејҸеҸҜд»ҘиҷҡжӢҹеҮәзұ»BпјҢйҒҝе…ҚиҖҰеҗҲгҖӮ
  • дёҺе…¶д»–жЁЎејҸж··еҗҲдҪҝз”ЁгҖӮд»ҺиҖҢжһ„йҖ еҮәжү©еұ•жҖ§жӣҙеҘҪзҡ„и®ҫи®ЎгҖӮ

дёәеҲӣе»әдёҖз»„зӣёе…іжҲ–зӣёдә’дҫқиө–зҡ„еҜ№иұЎжҸҗдҫӣдёҖдёӘжҺҘеҸЈпјҢиҖҢдё”ж— йңҖжҢҮе®ҡд»–д»¬зҡ„е…·дҪ“зұ»гҖӮ

дјҳзӮ№пјҡ

зәҰжқҹдёәйқһе…¬ејҖзҠ¶жҖҒпјҢеңЁдёҚеҗҢзҡ„е·ҘеҺӮдёӯпјҢеҗ„з§Қдә§е“Ғд№Ӣй—ҙеҸҜиғҪе…·жңүдёҚеҗҢзҡ„зӣёдә’дҫқиө–е…ізі»пјҢиҝҷдәӣдҫқиө–е…ізі»з”ұе·ҘеҺӮе°ҒиЈ…еңЁе…¶еҶ…йғЁеҜ№дәҺе·ҘеҺӮзҡ„дҪҝз”ЁиҖ…жқҘиҜҙдёҚеҸҜи§ҒгҖӮ

жү©еұ•йқһеёёе®№жҳ“гҖӮиӢҘиҰҒй’ҲеҜ№еҗҢдёҖдә§е“Ғж—Ҹз®ҖеҺҶж–°зҡ„з”ҹдә§зәҝпјҢеҸӘйңҖе®һзҺ°дә§е“Ғж—ҸдёӯжүҖжңүдә§е“ҒжҺҘеҸЈе№¶е»әз«Ӣж–°зҡ„е·ҘеҺӮзұ»еҚіеҸҜгҖӮ

зјәзӮ№пјҡ

дә§е“Ғж—Ҹжң¬иә«зҡ„жү©еұ•йқһеёёеӣ°йҡҫгҖӮеҰӮжһңдә§е“Ғж—ҸдёӯеўһеҠ дёҖдёӘж–°зҡ„дә§е“Ғзұ»еһӢпјҢеҲҷйңҖиҰҒдҝ®ж”№еӨҡдёӘжҺҘеҸЈпјҢе№¶еҪұе“Қе·Іжңүзҡ„е·ҘеҺӮзұ»гҖӮ

  • еҺҹеһӢжЁЎејҸпјҡзұ»дјјдәҺе·ҘеҺӮжЁЎејҸпјҢдёҚеҗҢеңЁдәҺеҺҹеһӢжЁЎејҸйҖҡиҝҮеӨҚеҲ¶дёҖдёӘзҺ°жңүеҜ№иұЎз”ҹжҲҗж–°еҜ№иұЎгҖӮ
